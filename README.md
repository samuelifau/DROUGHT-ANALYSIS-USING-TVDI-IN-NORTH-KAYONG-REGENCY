### TVDI Formula
TVDI = (LST - LST_min) / (LST_max - LST_min)

### Drought Classification
| TVDI Value | Drought Level |
|-----------|----------------|
| 0.0 â€“ 0.2 | Very Wet |
| 0.2 â€“ 0.4 | Wet |
| 0.4 â€“ 0.6 | Normal |
